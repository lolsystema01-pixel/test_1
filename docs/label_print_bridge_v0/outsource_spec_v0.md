# 印刷ブリッジ本体＋.lbx 外注仕様書 v0（スポット外注検討用）

**孤立部品**：ローカルWindows agent が **ラベルペイロード**を受け取り、**b-PAC** で **Brother TD-2350** に印刷する。
本書は発注用の最小仕様。内製コアとの境界は [adapter_contract_v0.md](adapter_contract_v0.md)。

## 1. スコープ

| 区分 | 内容 |
| --- | --- |
| 発注（外注） | ① 印刷ブリッジ本体（Windows agent：ペイロード→b-PAC→TD-2350）<br>② `.lbx`（ラベルレイアウト）を**サンプル見本準拠で新規作成** |
| 支給（内製済） | ラベルペイロード（§3）・送信フック・PDF・履歴/再印刷・バーコード枠ON/OFF |
| 対象外 | ラベル“内容”の変更（確定済）／帳票（配車表PDF・かご持出表PDF・GoDoor CSV）／他の孤立部品 |

## 2. 前提・環境

- **Windows**（b-PAC は Windows COM＝ローカル実行・クラウド不可）。
- **Brother TD-2350** ＋ b-PAC SDK。**機種抽象化**：プリンタ／ライブラリを差替えても §contract を満たすこと。
- 入力＝ペイロード契約、出力＝印刷。受け口はローカルHTTP `POST /print`（推奨）／ファイル投下／キューのいずれか。

## 3. 入力（ラベルペイロード＝確定）

[adapter_contract_v0.md §2](adapter_contract_v0.md) のJSON。要点：

- **数字のみ**。住所・氏名は載せない。
- 大ラベル＝`basket_code + " " + delivery_order`（例「西-5 1」）。
- 小ラベル＝`tracking_number`。

## 4. `.lbx` 新規作成（サンプル見本準拠）

現行に `.lbx` 実ファイルは無いため、**提供済みサンプル見本**から **P-touch Editor** で新規作成。

| 要素 | 仕様（見本準拠） |
| --- | --- |
| 大サイズ | **36**＝かご記号＋配達順 |
| 小サイズ | **12**＝問合番号 |
| 高さ | **約30mm** |
| 用紙幅 | 見本の "x"＝**未指定** → 実機で確定（**要確認**） |
| 配置 | 見本どおり（外注で正確な配置を調整） |
| バーコード枠 | **非表示（OFF）が既定**。ONにできる差込口だけ用意。ON時のみ問合番号で描画（**シンボロジーは配送伝票に合わせる**） |

差込フィールド：`basket_code`・`delivery_order`・`tracking_number`（＝ペイロードのキーに対応）。

## 5. 動作（印刷ブリッジ本体）

1. 受け口でペイロードJSONを受信（`items[]`）。
2. 各 item を `.lbx` の差込フィールドへ流し込み、b-PAC で TD-2350 へ印刷。
3. 受理を `200 OK`（HTTP）等で返す。実印刷の成否はagent側ログ/UIで管理。
4. `kind`（print/reprint/pdf）は記録用。reprint も同じ印刷経路でよい。

## 6. §8.5 配達前照合（スキャナ・要確認）

- 仕分け・配達前照合は**配送伝票の問合番号バーコードを読む**（我々はラベルにバーコードを刷らない）。
- スキャナを**配送伝票のバーコード**に合わせること（読み取り対象・シンボロジーの確認）。＝ **【人】確認事項**。

## 7. 受入条件

- [ ] ペイロードJSON（§3）を受けて、見本どおりのラベル（大＝かご記号＋配達順／小＝問合番号・数字のみ・約30mm）が TD-2350 から出る。
- [ ] バーコードは**既定非表示**。設定ONで枠に問合番号を描ける。
- [ ] 受け口（HTTP/ファイル/キュー）が [adapter_contract_v0.md §3](adapter_contract_v0.md) を満たし、内製コアから無改修で送信できる。
- [ ] `.lbx` が支給見本準拠（36/12/30mm）。用紙幅は実機確定。
- [ ] Windows・b-PAC 前提で動作（インストール手順付き）。

## 8. 【人】の残タスク（外注前に確定）

- ラベル**用紙幅**の確定（見本の "x"）。
- §8.5 照合スキャナを配送伝票バーコードに合わせる情報（対象・シンボロジー）。
- 提供済みサンプル見本（画像：36/12/30mm）を外注へ支給。
