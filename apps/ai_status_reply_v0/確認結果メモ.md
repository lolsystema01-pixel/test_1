## 備考・気づき

- PIIマスキングは **DBの SECURITY DEFINER 関数で源流強制**。サーバ(Hono)も Claude も氏名・詳細住所・連絡先を受け取らない。＝service_role不要・anonキーのみ。
- Claude APIキーは `ANTHROPIC_API_KEY`（サーバ環境変数）。SDK が環境から読む。コード・レスポンス・起動ログに出さない。
- モデルは `claude-opus-4-8`（環境変数で変更可）。単一メッセージ・非ストリーミング・max_tokens=1024。
- 動かす場所：PoCはローカルHono。Cloud Run デプロイ時も同コードで、キーは Secret Manager に。
- 範囲外：荷受人の認証ゲート（7.1）。本番化時に「問合番号＋簡易認証」を前段に必須。
- Claude 実呼び出し（自然文の中身）はキーを伴うため【人】が実施。本メモのステータス別表に記録する。
